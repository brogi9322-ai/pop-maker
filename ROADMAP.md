## Phase 1: 메인 에디터 완성 🔄

> 해커톤 데모 기준 완성 목표

### Sprint 1: 핵심 UX 기능 (P0)

- ⬜ Undo/Redo — Fabric.js 내장 히스토리 + Ctrl+Z/Y 단축키
- ⬜ 레이어 패널 — 드래그 재정렬, 잠금/숨기기, 이름 변경, 그룹화
- ⬜ 3패널 레이아웃 개선 — 좌: 템플릿/레이어 탭 | 중: 캔버스 | 우: 속성 패널
- ⬜ 캔버스 규격 프리셋 + 직접 입력 (A4, A5, 정사각형, 배너)

### Sprint 2: AI 생성 + 내보내기 (P1)

- ⬜ AI POP 생성 — 텍스트 프롬프트 → Claude API → Fabric.js JSON 렌더링
- ⬜ PNG 내보내기 최적화 — `canvas.toDataURL()` 사용, 3초 이내
- ⬜ PDF 내보내기 — jsPDF 연동
- ⬜ JSON 템플릿 내보내기/가져오기

### Sprint 3: 공유 + 마무리 (P2)

- ⬜ 공개 템플릿 공유 — `isPublic` 플래그 + 공유 링크 (`/template/{docId}`)
- ⬜ 모바일 미리보기 — 탐색/저장 전용 모바일 레이아웃
- ⬜ UI 폴리싱 — 온보딩 힌트, 에러 토스트, 로딩 상태
