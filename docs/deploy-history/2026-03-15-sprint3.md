## 구현 내용

| 기능 | 설명 |
|------|------|
| PNG 내보내기 | html2canvas scale:2, 날짜 포함 파일명, 성공 토스트 |
| PDF 내보내기 | unit:'mm' + px→mm 변환 (96dpi 기준) |
| JSON 내보내기/가져오기 | Blob 다운로드, FileReader 파싱, id 재채번, 10MB 제한 |
| Firebase Hosting | firebase.json + .firebaserc + deploy 스크립트 |
| 인라인 텍스트 편집 | 더블클릭 → contentEditable, Escape 취소 |
| 캔버스 경계 클램핑 | 드래그 시 요소 캔버스 밖 이탈 방지 |
| 사이즈 비례 이동 | 캔버스 크기 변경 시 요소 위치 비율 유지 |
| Ctrl+C/V | 요소 복사·붙여넣기 (+10px 오프셋) |
| Delete/Backspace | 선택 요소 삭제 |
| 에셋 패널 | SVG 아이콘 23개 (감기/콧물, 소아, 임산부, 약, 건강 카테고리) |
| 토스트 UI | 하단 중앙, 자동 dismiss, error/success 타입 |
| 디자인 시스템 | CSS 변수 + 다크모드 (data-theme="dark") |
| Firebase 미설정 알림 | 저장 시 toast.error로 안내 |
